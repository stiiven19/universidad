## 🚀 Proyectos Destacados

### Java

#### 🎯 Jaque4Caballos
- **Tecnología**: Java, JGraphT, GraphStream
- **Descripción**: Implementación de algoritmo de búsqueda informada A* para resolver el problema de jaque mate con 4 caballos
- **Características**: Función heurística personalizada, visualización gráfica, optimización de búsqueda

#### 🖥️ Evaluations
- **Tecnología**: Java Swing, Java 2D
- **Descripción**: Componentes de interfaz gráfica personalizados
- **Componentes**: GridSelect, Hyperlink, ojos que siguen cursor, barras de progreso

#### 🛠️ Talleres
- **B-BasicGUIApplications**: Interfaces gráficas básicas con Swing
- **C-AdvancedGUIApplications**: Interfaces gráficas avanzadas
- **D-GeneralApplications**: Aplicaciones completas
- **Graficos**: Programación gráfica y visualización

### Python

#### 📊 Estructuras de Datos (ds/)
- **Tecnología**: Python, GTK, Tkinter
- **Descripción**: Implementación completa de estructuras de datos clásicas
- **Estructuras**: Listas enlazadas, pilas, colas, árboles, grafos
- **Aplicaciones**: 
  - `grafos_app`: Manipulación y visualización de grafos con algoritmo de Floyd
  - `colegio_app`: Sistema de gestión escolar con interfaz GTK

## 📋 Instalación General

1. **Clonar el repositorio**:
   ```bash
   git clone https://github.com/stiiven19/universidad.git
   cd universidad
   ```

2. **Requisitos por proyecto**:
   - **Java**: JDK 8+, NetBeans IDE
   - **Python**: Python 3.7+, GTK 3, PyGObject

3. **Navegar al proyecto específico**:
   ```bash
   # Para proyectos Java
   cd Java/[nombre-proyecto]
   
   # Para proyectos Python
   cd python/[nombre-proyecto]
   ```
